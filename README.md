# A rrule field for Nova apps

This package contains a Nova Card for agenda application.

## Requirements

- Laravel Nova
- Laravel Agenda application
  - App\Models\Agd\Room
  - App\Http\Resources\Agenda\RoomResource

## Installation

Install

```bash
composer require mmerlijn/nova-show-agenda-card
```
